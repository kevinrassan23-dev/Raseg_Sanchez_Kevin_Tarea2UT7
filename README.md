# Aplicación de recetas canarias con navegación con comandos por voz:
En este repositorio se aloja una aplicación web que habla sobre las islas canarias
y su gastronomía a modo de introducción, el contenido de esta aplicación son 
5 recetas con los platos más famosos y conocidos de todo el archipiélago. 

Por otra parte, la aplicación cuenta con un sistema de navegación por comandos de voz
integrando una interfaz natural de usuario "NUI" que se encargará de realizar una serie
de acciones, desde subir y bajar la pantalla hasta navegar entre las diferentes secciones
y recetas documentadas.

## Comandos de voz documentados:
| Comando de voz | Acción |
|---|---|
| "para el chozo" | Navega a la página principal |
| "para abajo" | Scroll hasta el final de la página |
| "para arriba" | Scroll hasta el inicio de la página |
| "papas arrugadas" | Navega a la receta de Papas Arrugadas |
| "ropa vieja" | Navega a la receta de Ropa Vieja |
| "potaje de berros" | Navega a la receta de Potaje de Berros |
| "carne mechada" | Navega a la receta de Carne Mechada |
| "gofio canario" | Navega a la receta de Gofio Escaldao |


> NOTA: Se deben conceder permisos de audio en la aplicación.
